## Install

In your `Cargo.toml`,

```
pir = { git = "https://github.com/raviqqe/pir", branch = "master" }
pir-fmm = { git = "https://github.com/raviqqe/pir", branch = "master" }
```

## Features

- Currying
- Nominal typing
- Open variant type
- Foreign function interface
